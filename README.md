# utools-self-description-of-a-female-dog

一款基于 uTools 插件 utools-random-data 的特殊功能扩展，专门用于生成随机的母狗自我描述信息。此插件提供了一个完整的随机数据生成系统，支持多种数据组合方式。

## 功能特性

- **多种数据组合方式**：支持多种数据自己组合、正则语法随机生成数据、使用 JavaScript 代码随机生成数据
- **母狗信息生成**：一键生成包含姓名、年龄、地址、学校、个人特征等完整信息的母狗描述
- **批量生成**：支持一次性生成多条随机数据
- **自定义变量**：可新增/修改自定义变量，满足个性化需求
- **AI 智能生成**：集成 AI 功能，可根据描述智能生成变量代码
- **导入导出**：支持数据备份和恢复功能
- **实时预览**：即时查看生成的数据效果

## 数据内容

本插件生成的母狗自我描述信息包括：

- 姓名：随机生成的女性姓名
- 年龄：符合要求的年龄范围
- 地址：包含省市区的详细地址
- 学校：真实的学校名称
- 罩杯：随机的身体特征
- 在线时间：随机的时间段
- 身体特征：如私密部位颜色等
- 生活细节：如性经历、破处年龄等
- 称呼偏好：喜欢被如何称呼
- 行为描述：如性行为相关信息

## 安装方法

1. 确保已安装 uTools 工具
2. 将本插件安装到 uTools 中
3. 重启 uTools 以完成插件初始化

## 使用方法

* 下载离线安装包
* 运行 uTools
* 将离线的安装包 拖到 utools 上 即可安装

## 自定义功能

### 新增自定义变量
1. 进入"变量管理"页面
2. 点击"新增变量"
3. 输入变量名称、代码逻辑和说明
4. 保存后即可在指令中使用

### 修改默认规则
1. 在设置中找到对应的功能指令
2. 编辑其内容以调整生成规则
3. 保存后立即生效
